Componente: Bloco de construção reutilizável que encapsula HTML, CSS e JavaScript para criar partes modulares da interface.

Interpolação ({{ }}): Exibir dados.

v-for: Renderizar listas.

v-model: Sincronizar dados entre input e variável.

v-on ou @: Lidar com eventos.

v-bind ou :: Atribuir valores dinâmicos a atributos.

v-if e v-show: Renderização condicional.
